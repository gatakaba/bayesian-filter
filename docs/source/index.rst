Bayesian Filter
===================

ベイジアンフィルタとは?
----------------------

ベイジアンフィルタとは観測値に基づき状態量を更新する情報処理手法の一つです。

特色としては状態量を確率密度関数により表し、新たに観測されたデータ(観測値)とベイズの定理を用いて状態量の確率密度関数を更新する点です。

確率的に表された状態量と状態遷移関数を用いることによって、未来予測や欠損値推論など、観測されてないデータを確率的に推測することができます。

ベイジアンフィルタには様々な手法があります。

最も良く知られているのはカルマンフィルタでしょう。

カルマンフィルタは状態遷移関数と観測関数が線形かつガウシアンノイズという仮定で推定が行われます。

カルマンフィルタは状態量の性質が線形変換によって不変であるという性質を用いて比較的小規模な計算量で推論を行う事ができるので、多くの線形モデルにおいてよく使われます。

一方非線形モデルにおけるベイジアンフィルタの計算方法としてExtened Kalman Filter,Unscented Kalman Filter,Particle Filter などがよく使われます。

Extended Klaman FilterやUnscnented Kalman Filterは特定の点を用いて確率密度関数を近似する手法です。

Particle Filterはモンテカルロ法を用いて確率密度関数を近似する手法です。

詳しい説明はここでは割愛しますが、前者は計算量が比較的少なく済む反面推定誤差が大きい性質を持ち、前者は推定誤差が少なく済む反面計算量が大きくなりがちで
リアルタイム推定が困難になる場合があるという性質を持ちます。

どの手法を用いるかはモデルや問題の性質を加味して慎重に決めて下さい。
